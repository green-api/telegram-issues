# GREEN API Issues (Telegram)

## How to help

If you notice a [bug](https://github.com/green-api/telegram-issues/issues/new?template=bug_report.md) or have any [suggestions for improvement](https://github.com/green-api/telegram-issues/issues/new?template=feature_request.md), you can create a thread in the `Issues` section.

## Как помочь

Если вы заметили [ошибку](https://github.com/green-api/telegram-issues/issues/new?template=bug_report.md) или у вас есть [предложения по улучшению](https://github.com/green-api/telegram-issues/issues/new?template=feature_request.md), вы можете создать тему в разделе `Проблемы`.
